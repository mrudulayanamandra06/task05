## Objective
Capture live network traffic using Wireshark and identify basic protocols.
## Tools
Wireshark
Windows
Active network interface
## Steps
Installed Wireshark.
Started capture on the active interface.
Generated traffic by using ping.
Stopped capture after one minute.
Applied filters like tcp, udp, dns.
Saved the capture as a .pcap file.
## Protocols Found
TCP
UDP
DNS / SSDP
## Outcome
Basic understanding of packet capture and protocol analysis using Wireshark.
